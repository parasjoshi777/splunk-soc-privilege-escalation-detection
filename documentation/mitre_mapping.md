# MITRE ATT&CK Mapping

## T1548 – Abuse Elevation Control Mechanism
- Detected through repeated sudo executions in short time windows
- Indicates potential privilege escalation abuse

## T1110 – Brute Force
- Behavior-based inference through rapid repeated sudo attempts
- Suggests password guessing or misuse

## Coverage Summary
This project provides detection coverage for privilege escalation and
credential abuse techniques using SIEM-based behavioral analytics.
