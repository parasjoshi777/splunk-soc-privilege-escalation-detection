# SOC Privilege Escalation Dashboard

## Purpose
This dashboard provides SOC analysts with visibility into abnormal
privilege escalation behavior detected through sudo activity.

## Panels Included
1. Excessive sudo attempts (Alert View)
2. Severity-based privilege escalation
3. Investigation view with executed commands
4. Activity trends over time

## Analyst Usage
- Identify suspicious privilege escalation quickly
- Review affected host and time window
- Inspect executed commands for intent
- Prioritize incidents based on severity
